module Stage : sig .. end
A stage is an effectful computation that takes an environment and produces zero or more results.
type ('r, 'a, 'b) t = 'r -> 'a -> 'b list Async.Std.Deferred.t
('r, 'a, 'b') t is type for stages that take an environment of type 'r, and argument of type 'a, and produces values of type 'b.
val compose : ('r, 'b, 'c) t -> ('r, 'a, 'b) t -> ('r, 'a, 'c) t
compose s2 s1 is a stage that, when run, will run the s2 stage first, and pass any results into the s1 stage. The enviornment will be threaded through both stages in the same order as the stages are run.
val (>=>) : ('r, 'a, 'b) t -> ('r, 'b, 'c) t -> ('r, 'a, 'c) t
s1 >=> s2 is equivalent to compose s2 s1
val (<=<) : ('r, 'b, 'c) t -> ('r, 'a, 'b) t -> ('r, 'a, 'c) t
s2 <=< s1 is equivalent to compose s2 s1
val combine : ('r, 'a, 'b) t -> ('r, 'a, 'b) t -> ('r, 'a, 'b) t
combine s1 s2 is a stage that, when run, will pass the environment and argument to both stages. The result of combine s1 s2 is the result of s1 and s2, concatenated.
val (<|>) : ('r, 'a, 'b) t -> ('r, 'a, 'b) t -> ('r, 'a, 'b) t
s1 <|> s2 is equivalent to combine s1 s2.
val local : ('r1 -> 'r2) -> ('r2, 'a, 'b) t -> ('r1, 'a, 'b) t
local f transforms a stage that expects an evironment type of 'r2 into a stage that expects an environment type of 'r1. It does this by using f to recover a value of type 'r2 from a value of type 'r1.
val run : ('r, 'a, 'b) t ->
'r -> 'a Async.Std.Pipe.Reader.t -> 'b Async.Std.Pipe.Reader.t
run s t turns the stage into a function that will transform a Pipe.Reader.t that prodcues values of type 'a to a Pipe.Reader.t that produces values of type 'b. In essence, run s t lifts the stage into a pipe transformer, and uses the t value as the environment on each invocation of the stage.
