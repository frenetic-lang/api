Exception sexp converters that are Unix-specific. Handles Unix.Unix_error.

Write let () = Sexplib_unix.Sexplib_unix_conv.linkme in your program to ensure that the code in this module is run, i.e. the unix-specific exception converters are added. This is already done by Core.Std, so any application that uses Core need not worry about this module at all.
